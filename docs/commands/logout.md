# `logout`

## Introduction

The `logout` command clears the API key from your device.

## Usage

```bash
redocly logout [--help] [--version]

redocly logout --version
```

## Options

Option | Type | Description
-- | -- | --
--help | boolean | Show help.
--version | boolean | Show version number.
